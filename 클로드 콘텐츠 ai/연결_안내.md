# 연결 안내 (혜미용 — 5분 컷)

## 1. Claude Chrome 확장 연결 (게시 대행에 필수)

1. **Chrome을 연다** (컴퓨터의 크롬 브라우저)
2. 크롬 웹스토어에서 **"Claude in Chrome"** 확장을 설치한다
   (이미 설치돼 있으면 크롬 오른쪽 위 퍼즐조각 아이콘 → Claude 고정)
3. 확장 아이콘을 눌러 **Claude 계정으로 로그인**한다
4. Claude(Cowork) 대화창에서 "크롬 연결됐는지 확인해줘"라고 말하면 내가 확인

> 연결되면: 블로그·스레드 글 읽기(스타일 분석) + 승인된 글 게시 대행이 가능해짐.
> 원칙: 네이버/스레드 **로그인은 항상 혜미가 직접** — Claude는 비밀번호를 다루지 않음.

## 2. 티스토리 개설 (수익화용 두 번째 블로그)

1. https://www.tistory.com 접속 → **카카오 계정으로 시작**
2. 블로그 이름·주소(예: eunoia9496.tistory.com) 정하기 — 네이버와 통일하면 관리 편함
3. 개설 후 주소를 Claude에게 알려주면 `channel_config.yaml`에 등록
4. (수익화) 글 10개 이상 쌓이면 구글 애드센스 신청 — 이건 나중 단계

## 3. 연결 후 첫 게시 흐름

`초안/테스트1호_2026-07-12.md` 승인("1번 승인" 식으로 채팅) → Claude가 Chrome으로
채널별 게시 → 발행 장부 기록. 이후부터는 "마시땅 글 써줘" 한마디로 반복.
