# AI 인플루언서 설계 — 내 얼굴 없이 인스타 운영 #콘텐츠

> Fable 확정 설계. 실행은 Sonnet이 이 문서만 보고 진행. 상위 설계: [[ai공장짓기/설계노트/공장별/공장-이미지AI인간_공장|이미지·AI인간 공장]]

## 1. 도구와 연결 (혜미 몫, 1회 15분)
1. **Higgsfield 연결**: 클로드 설정 → Connectors → 커스텀 커넥터 추가 → `https://mcp.higgsfield.ai/mcp` → 로그인 (구독 필요 — 크레딧제).
2. **인스타 새 계정 개설** (계정 생성·비번은 항상 혜미 직접 — 절대규칙).
3. 연결 후 아무 세션에서 "히그스필드로 테스트 이미지 1장 만들어줘"로 작동 확인.

## 2. 같은 얼굴 유지하는 법 (핵심 기술 규칙)
1. **페르소나 카드 1장이 진실 원본** — 얼굴·헤어·체형·패션톤·조명 문구를 고정 텍스트로 저장(아래 3번). 매 생성마다 이 문구를 그대로 사용.
2. **첫 확정 이미지를 레퍼런스로 고정** — 마음에 드는 1장이 나오면 그 이미지를 모든 후속 생성의 참조 이미지로 첨부. (설계노트의 "재생성 시 seed 변경 필수" 규칙은 반려 시에만)
3. **조명·색감 문구는 복붙** — "soft window light, warm beige tone" 같은 문구를 페르소나 카드에 박아두고 변형 금지.
4. **저작권/초상권 게이트**: 실존 인물 닮게 만들기 금지, 생성 후 실존인물 유사성 자가검사(공장 설계 rights_check 그대로).

## 3. 페르소나 카드 (혜미가 채울 틀 — Sonnet 인터뷰로 10분)
새 Sonnet 세션에 복붙:
```
클로드 콘텐츠 ai/AI인플루언서_설계.md 3번의 페르소나 카드를 나를 인터뷰해서 채워줘.
질문 하나씩: ①콘셉트(육아맘/리빙/뷰티 중) ②나이대·분위기 ③패션 톤 ④계정 이름 후보 3개
⑤mo,on 사업과 연결할지. 완성되면 이 문서에 카드 섹션으로 추가하고 결과를 보여줘.
```
카드 필드: 이름 / 콘셉트 한 줄 / 외모 고정 문구(영문) / 조명·색감 고정 문구(영문) / 말투 / 금지 소재

## 4. 운영 규칙 (Fable 확정)
- **AI임을 숨기지 않기** — 프로필에 AI/버추얼 표기. 이유: 들통 리스크 제거 + AI 인플루언서 자체가 차별점이며 협찬주도 선호 추세.
- 주 3회 발행(콘텐츠 공장 스케줄에 편입), 게시는 승인 후 Chrome 대행 — 기존 게시봇 규칙 그대로.
- 성장 90일 전까지 수익 장치 안 붙임 (신뢰 먼저 — [[1. Projects/수익화_로드맵_2026H2|수익화 로드맵]] M3).

## 5. 생산 파이프라인 (러너 이식 전 임시 운영)
주제 선정(콘텐츠 공장 공유) → 페르소나 카드 문구로 Higgsfield 생성 3장 → 혜미 1장 선택 → 캡션 작성(Sonnet, 작업스타일 준수) → 승인 → 게시. 회당 혜미 몫: 선택+승인 2분.
