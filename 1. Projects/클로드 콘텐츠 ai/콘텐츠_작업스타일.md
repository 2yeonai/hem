# 혜미 콘텐츠 작업 스타일 (지시: "이 컨텐츠 일하는 스타일도 남겨줘")

> 어느 세션/모델이든 콘텐츠 작업 전에 이 파일을 먼저 읽을 것.
> 1부는 혜미가 일하는 방식(확정), 2부는 글 스타일 프로필(골든셋 수집 후 채움).

## 1부 — 일하는 방식 (2026-07-12 혜미 회신 기준, 확정)

1. **역할 분담**: 혜미는 주제만 던진다. 조사→분석카드→초안→형식검사까지는 공장(AI)이 하고, 혜미는 **승인 클릭**과 **사람만 아는 정보 입력**(직접 찍은 사진, 실제 경험담)만 한다.
2. **기초자료는 하나, 채널은 여럿**: 같은 분석카드 + 혜미가 쓴 글 스타일 + 혜미가 직접 찍은 사진에서 모든 채널(네이버/티스토리/스레드)이 파생된다. 채널별로 새로 만들지 않는다 — 형식만 바꾼다.
3. **수익화 전제**: 네이버·티스토리 둘 다 운영하는 이유가 수익화(애드포스트/애드센스). 글 구성 시 검색 유입·체류시간을 의식하되, 광고티 나는 글은 혜미 스타일이 아님(2부 프로필로 보정).
4. **승인 없이 게시 없음**: 어떤 경우에도 혜미의 승인완료 없이 외부 게시 금지 — 공장 구조(게시봇 이중방어)로 강제됨. "진짜 자동화" = 승인 전 전 과정 자동 + 승인 후 게시 대행.
5. **발행 리듬**: 주 3회, 5시(오전/오후·요일 미확정). 발행일에 초안이 미리 준비돼 있고 혜미는 승인만 하는 흐름이 목표.
6. **기록 습관**: 게시할 때마다 HANDOFF의 발행 장부에 남긴다. 반려할 때는 사유(형식/내용/제약위반)를 골라주면 공장이 알맞은 단계로 되돌아간다.

## 2부 — 글 스타일 프로필 (골든셋 수집 대기 — 빈칸)

> 채우는 방법: Chrome 확장 연결 후 아래 계정에서 대표 글 3~5개 수집, 또는 혜미가 복붙.
> - 네이버: https://m.blog.naver.com/eunoia9496 ("eunoia: 유노이어")
> - Threads: @maasittang / @2yeon_sz

| 항목 | 값 (수집 후 기입) |
|---|---|
| 톤/말투 | (예: 존댓말/반말, 이모지 사용 여부) |
| 블로그 글 평균 길이 | |
| 문단 구성 습관 | (예: 소제목, 사진 배치 위치) |
| 해시태그 개수/스타일 | |
| 자주 쓰는 표현 | |
| 피하는 표현(금지어 후보) | |
| 사진 스타일 | 직접 촬영 (혜미 제공 — AI 생성 아님) |
| 스레드 글 호흡 | (한 게시물 길이, 연결글 개수) |

수집·분석이 끝나면 이 표의 값을 `scripts/handlers.py`의 형식게이트 조건테이블(글자수/금지어/해시태그)에 반영할 것.
