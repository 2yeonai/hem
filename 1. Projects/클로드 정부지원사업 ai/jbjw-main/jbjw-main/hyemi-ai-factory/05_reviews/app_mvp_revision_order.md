# app_mvp_revision_order.md — 앱 MVP 재작업 지시서

## 1순위 — 신뢰를 무너뜨릴 수 있는 것

| # | 항목 | 지시 | 담당 | 완료 기준 |
|---|---|---|---|---|
| 1-1 | 실제 공고 1건 실전 투입 | 사용자가 실제 공고 원문·서류로 새 프로젝트 생성 → 전체 흐름 1회전 → 오작동 기록 | **사람+앱** | 실전 프로젝트 1개 + 오작동 목록 |
| 1-2 | 브라우저 실사용 테스트 | 사람이 /new 폼을 직접 채워 제출 — 입력 UX 결함 수집 | **사람** | 사용 소감·결함 메모 |
| 1-3 | 점수 착시 방지 | judge 탭 상단에 "이 점수는 감점 요인 잔량 지표이지 심사 예측이 아님" 문구 이미 있음 → 실사용 후 위치·강도 재점검 | AI | 실사용 피드백 반영 |
