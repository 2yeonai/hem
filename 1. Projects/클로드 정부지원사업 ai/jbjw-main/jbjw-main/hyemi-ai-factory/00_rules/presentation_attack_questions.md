# presentation_attack_questions.md — 심사위원 공격 질문 은행

발표 있는 공고는 아래 질문 전체에 대한 답변을 준비해야 Q&A LOCK 가능.

## 필수 10대 질문

1. 왜 이 아이템인가?
2. 왜 지금 필요한가?
3. 왜 신청자가 해야 하는가?
4. 기존 방식과 무엇이 다른가?
5. 수행기간 안에 가능한가?
6. 예산이 왜 필요한가?
7. 단순 구매나 외주 의존 아닌가?
8. 고객 또는 수혜자는 어떻게 확보할 것인가?
9. 실패하면 어떻게 보완할 것인가?
10. 공고 목적과 어떻게 맞는가?

## WHY 3종 방어 구조

| 질문 | 답변 재료 |
|---|---|
| 왜 이 아이템인가 | 문제의 구체성 + 기존 대안의 한계 |
| 왜 지금인가 | 공고 취지 + 환경 변화 + 방치 시 손실 |
| 왜 신청자인가 | 경험 + 보유 자원 + 현장 이해도 + 고객 접점 |

## 추가 공격 질문 (상황별)

- 누가 돈을 내는가 / 수혜 구조는 무엇인가?
- 성과를 어떻게 측정하는가?
- 그 성과목표는 현실적인가? 근거는?
- 금액 산정 근거는? 예산을 30% 줄이면?
- 증빙자료가 있는가? 지금 보여줄 수 있는가?
- 유사 서비스가 이미 있지 않은가?
- 신청자가 직접 만들 수 있는가, 외주인가?
- 수행기간 후에도 지속 운영이 가능한가?

## 모르는 질문 대응 원칙

- 아는 척하지 않는다. 확정되지 않은 것을 확정처럼 말하지 않는다.
- 안전 문장: "현재 단계에서는 제출한 수행범위 안에서 먼저 검증하고, 결과를 바탕으로 다음 단계에서 보완하겠습니다."

## 답변 평가 기준

- 질문에 직접 답했는가 / 장황하지 않은가 / 과장이 없는가 / 증빙을 언급했는가 / 예산·산출물을 연결했는가 / 발표자가 실제로 말할 수 있는가
