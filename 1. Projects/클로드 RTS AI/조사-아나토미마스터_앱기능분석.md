# 아나토미마스터 앱 기능 분석 — RTS 복습 서비스 벤치마크

> 목적: RTS 복습·실기 연습 서비스에 넣을 기능의 기준 앱. 출처: Google Play 공식 설명·리뷰(2026-07-16 조사).
> 벤치마크(benchmark) = 잘 만든 기존 제품을 기준 삼아 비교·참고하는 것.

## 1. 앱 개요

- (주)용감한컴퍼니(이파마스터) 제작, "국내 최초 3D 해부학 암기앱", 평점 4.8(리뷰 621개)
- 대상: 필라테스 강사·트레이너·물리치료사 — 우리 P2·P3 페르소나와 겹침
- 핵심 한 줄: **"해부학이 입 밖으로 툭 나올 때까지 암기시킨다"**

## 2. 기능 목록과 RTS 서비스 대응

| 아나토미마스터 기능 | 내용 | RTS 서비스 반영 |
|---|---|---|
| 3D 해부학 | 근육 위치+질감·결 모양까지 3D 이미지, 모션으로 움직임에 쓰이는 근육 확인 | F2 확장 목표. 결 방향은 F4(압력·근섬유 방향)와 직결 |
| 암기카드 | 부위별 이미지↔명칭 매칭, 원어/구용어/신용어 + 기시·정지·작용 한 번에, **명칭 가리기**, 좌우 스와이프 | F1·F5 카드에 "가리기 모드" 추가 (Muscle 스키마의 3표기 필드 활용) |
| 커리큘럼 | 뼈·랜드마크→근육→기시정지작용 순서로 하루 단위 분량 제공 | 복습 캘린더(F10)에 "일일 커리큘럼" 개념 참고 |
| 퀴즈·모의고사 | 객관식/주관식/빈칸 채우기, 모의고사 결과로 부족한 파트 진단 | F8에 빈칸 채우기 유형 추가, F10 약점 진단과 동일 사상 |
| 암기학습 관리 | 암기리포트(오늘/총 암기량·시간), 출석체크, 푸시 알림, 배지 | F10 확장: 복습 리포트 + 연속 학습(스트릭) + 배지 |
| 암기노트·오답노트 | 헷갈리는 카드 저장, 주제별 오답 반복 | F10 오답노트와 동일 — 카드 북마크 기능 추가 |
| 검색 | 키워드로 용어+이미지 즉시 검색 | 신규: 전체 엔티티 검색(검사·근육·테크닉 통합) — 여정 A의 "시술 전 5분"에 필수 |

## 3. 우리가 가져올 것 / 안 가져올 것

**가져온다**: 명칭 가리기 카드, 빈칸 채우기 퀴즈, 통합 검색, 학습 리포트·스트릭·배지, 근육 결 방향 시각화(F4와 연결).

**안 가져온다(차별점 유지)**: 아나토미마스터는 "지식 암기"가 끝이다. 우리는 검사→치료→재검사 **체인 훈련**이 핵심이므로, 암기 기능은 체인의 보조로만 넣는다(기획서 §11-1 원칙).

## 4. 반면교사 (리뷰에서)

- "뼈를 하나씩 떼어 보여주니 전체 구조 파악이 어렵다" → F2에서 전신/부위 맥락 뷰 먼저, 개별 근육은 그 안에서 하이라이트
- "이미 본 카드 구분이 안 된다" → 카드에 열람/암기완료 상태 표시(F1에 이미 반영됨) 필수 유지
- "부위가 몸 어디에 있는지 표시가 없다" → 카드마다 위치 미니맵 필요
