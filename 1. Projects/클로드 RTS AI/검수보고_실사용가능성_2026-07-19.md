# RTS 복습앱 실사용 검수 보고 (2026-07-19, Fable 5) [hyemi]

혜미 지시: "실사용 가능한 설계인지, 마사지 공부용이니 안 되지 않게 무조건 실행되게 검수해줘."
**문서만 읽지 않고 실제로 앱 서버를 띄워 브라우저에서 동작을 확인한 결과다.**

## 결론 먼저

**실사용 가능하다.** 앱은 실제로 켜지고, 96장 카드가 다 열리고, 검사에서 테크닉까지 눌러서 이어지고, 퀴즈도 돈다.
공부 중 잘못된 내용을 외울 뻔한 오류 1건을 찾아 고쳤다. 다만 **폰에서는 아직 못 연다**(아래 3번) — 마사지 공부용이면 이게 제일 아쉬운 지점이다.

## 실제로 확인한 것 (전부 실행 검증, 문서 인용 아님)

| 항목 | 결과 |
|---|---|
| 앱 서버 기동 + 주요 파일 | 전부 정상(HTML·JS·데이터·3D자산·소책자 사진 모두 200 응답) |
| 카드 데이터 | 96장(검사 35 + 테크닉 61) 파싱 실패 0건 |
| 검사 카드 필수 내용 | 35장 전부 방법·정상기준·양성판단 보유 (누락 0) |
| 체인 연결(검사→근육→테크닉→재검사) | 검사 35개 중 34개가 눌러서 이동 가능, 검사당 평균 4.4개 링크 |
| 실기 지시 품질 | 자세 → 방법 5단계 → 구두 지시 → 포인트까지 실제로 따라 할 수 있는 수준 (견갑거근 ART 실물 확인) |
| 퀴즈 | 191문제 자동 출제, 실제로 문제 풀이 화면까지 정상 진행 |
| 소책자 손 위치 사진(F3) | 테크닉 61장 중 38장 보유, 실제 이미지 정상 표시(900px) |
| 3D 근육 | 데이터 정확함 — 견갑거근 근육 메시가 마네킹 목~어깨 높이에 정확히 위치, 강조 색도 목·상부몸통만 주황으로 올바름 |

## 찾아서 고친 것 — 잘못 외울 뻔한 내용 1건

**"색이 깊다"는 강의 녹음을 글로 바꾸는 과정에서 생긴 오류였다.**
강사는 "**관계가** 깊다"라고 말했는데("호기와 관계가 깊습니다") 녹취 파일에 "색이 깊다"로 잘못 적혔고, 카드가 그걸 그대로 옮겨 적었다. 원본 녹취(3.24)에서 같은 오류가 3곳 더 확인돼 추측이 아니라 확정이다.

- 문제의 심각성: 이 문장이 **퀴즈 문제로까지 출제되고 있었다**(외복사근 문제에 그대로 노출됨) — 즉 반복 학습으로 틀린 표현을 외우게 되는 구조였다.
- 조치: `테크닉_외복사근.md` 5곳 + `테크닉_내복사근.md` 1곳, 총 6곳을 "관계가 깊다"로 수정. 데이터 재생성(96장 실패 0) 및 sentinel 검사 통과.
- 같은 유형의 녹취 오류를 96장 전체에서 추가 검색했으나 **이 2장 외에는 없었다** — 카드 작성 품질 자체는 좋다.

## 실사용 전에 알아야 할 한계 3가지 (버그 아님, 사실 그대로)

**1. 테크닉 61장 중 17장은 손으로 하는 시술 단계가 없다.**
대둔근·복직근·햄스트링 3형제·내전근 등 주로 하체·몸통 근육이다. 이 카드들은 촉진(만져서 찾기)·평가·금기·임상 포인트는 있지만 ART/MET 시술 순서가 없다 — **강의에서 그 부분을 다루지 않았기 때문이고, 없는 걸 지어내지 않은 것이라 오히려 정직한 상태다.** 다만 "이 근육 시술법 보려고 열었는데 없네"가 될 수 있으니 미리 알고 있어야 한다.
- 시술 단계 보유 현황: ART 44장 / MET 20장 / 촉진·금기는 61장 전부 있음.

**2. 폰에서 못 연다 (마사지 공부용이면 이게 제일 아쉬움).**
앱 서버가 "이 컴퓨터에서만 접속 허용"으로 잠겨 있다(`server.py`의 `127.0.0.1`). 실기 연습하면서 폰으로 손 위치 사진을 보려면 이 한 줄을 `0.0.0.0`으로 바꾸면 같은 와이파이의 폰에서 열린다.
- 대신 알아둘 점: 그렇게 바꾸면 **같은 와이파이에 접속한 다른 기기에서도 이 앱이 보인다**(집 와이파이면 사실상 가족만). 공부 카드라 민감한 내용은 아니지만, 바꿀지는 혜미가 정할 일이라 이번엔 건드리지 않았다. 원하면 한마디만 하면 바꾼다.

**3. 3D 화면이 실제로 어떻게 보이는지는 아직 아무도 눈으로 못 봤다.**
데이터는 정확한 걸 이번에 확인했지만(근육 위치·강조 색 검증 완료), 검수용 브라우저가 3D 그림을 그리지 못하는 환경이라 "보기에 자연스러운가"는 판단 못 했다. **혜미가 PC에서 한 번 열어보면 5초 만에 확인된다** — 3D 탭에서 근육이 사람 몸의 엉뚱한 위치에 붙어 있지 않은지만 보면 된다.
