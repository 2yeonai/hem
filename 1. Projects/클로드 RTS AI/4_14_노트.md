## SECTION 1 | 경추 가동성 검사 개요

**💡 핵심 한 줄**
목 검사는 양적 기준 + 질적 기준 둘 다 본다 — 숫자보다 **좌우 비교 + 움직임의 질**이 핵심.

**📖 검사 구성**

| 번호 | 검사 | 기준 |
|---|---|---|
| 1 | 굴곡 (Flexion) | 손가락 2개 들어가는지 |
| 2 | 신전 (Extension) | 45도 각도 + 코끝·이마·턱 수평 |
| 3 | 회전 (Rotation) | SCM 수직 + 쇄골 절반까지 도달 |
| 4 | 가쪽굽힘 (Lateral Flexion) | 45도 + 목과 머리 함께 움직임 |

**📝 기록 방법**

| 기호 | 의미 |
|---|---|
| − | 음성 (정상) |
| + | 양성 (제한) |
| ++ | 통증 있을 때 무조건 |

## SECTION 2 | 굴곡 검사 (Cervical Flexion Test)

**💡 핵심 한 줄**
손가락 2개 들어가면 정상 — 상부경추 굴곡이 잘 나오는지 확인.

**📖 검사 기준**

| 항목 | 기준 |
|---|---|
| 자세 | 앉아서 |
| 방법 | 턱 당겨서 숙이기 |
| 정상 | 손가락 2개 들어가는 공간 확보 |
| 양성 | 공간 부족 or 통증 |
| 추가 | 45도 각도 나오는지 (척추·목 각도) |

**⚠️ 주의**
상부경추 굴곡 제한 → 하부경추가 더 많이 움직이려 함 → 하부경추 통증 유발

## SECTION 3 | 신전 검사 (Cervical Extension Test)

**💡 핵심 한 줄**
코끝·이마·턱 수평 + 귀·하악 수평 — 상부경추 신전 제한 여부 확인.

**📖 검사 기준**

| 항목 | 기준 |
|---|---|
| 자세 | 앉아서 |
| 양적 기준 1 | 이마·코끝 수직으로 수평 |
| 양적 기준 2 | 귀·하악 수평 |
| 양적 기준 3 | 턱·목적·흉골병 일직선 |
| 질적 기준 | 칼주름 생기면 정상 (칼주름 = 상부경추 신전 일어남) |

**📖 상부경추 신전 제한 패턴**

| 상황 | 의미 |
|---|---|
| 라인이 이렇게 꺾이면서 들어가고 다시 내려가는 모습 | 상부경추 신전 제한 |
| 더 많이 움직이려다 목이 꺾임 | 하부경추 과부하 |
| 신전했다가 못 돌아오는 경우 | 제한 |
| 갔다가 힘이 풀려서 확 넘어가는 경우 | 제한 |

> "이렇게 되면 하부경추가 아파요. 더 많이 움직이려고 하다보니까 원인은 상부경추를 못 움직여서."

## SECTION 4 | 회전 검사 (Cervical Rotation Test)

**💡 핵심 한 줄**
뒤에서 좌우 비교 먼저 → 앞에서 양적·질적 기준 확인 — 주시안 사용해서 정확하게.

**📖 검사 순서**

```
STEP 1. 뒤에서 → 좌우 비교 (어느 쪽이 더 잘 되는지)
STEP 2. 앞에서 → 양적·질적 기준 확인
```

**📖 양적 기준**

| 기준 | 내용 |
|---|---|
| SCM 수직으로 서야 함 | SCM이 기울어지면 제한 의심 |
| 코·턱 이은 선이 쇄골 절반까지 도달 | 코만 가거나 턱만 가면 머리 기울어진 것 |

> "코와 턱을 이은 선이 그대로 가는지 — 근데 보통 이렇게 돼요. 코만 가."

**📖 질적 기준**

| 기준 | 내용 |
|---|---|
| 머리가 팽이처럼 축회전 | 기울어지면 안 됨 |
| 눈높이 변화 없어야 함 | 쳐지거나 머리 기울어지면 제한 |
| 프로트랙션 동반 없어야 함 | 회전 시 머리 앞으로 빠지면 제한 |
| 이마면으로 움직여야 함 | 다른 면으로 벗어나면 제한 |

**📖 주시안 (Dominant Eye) 활용**

- 검사자가 주시안으로 대상자 머리 중앙을 봐야 정확함
- **주시안 찾는 법:** 손가락으로 원 만들고 한쪽 눈씩 윙크 → 가만히 있는 눈 = 주시안
- 주시안이 대상자 머리 가운데로 가게 해서 깐깐하게 보기

> "그냥 해도 되는데 이런 게 있다 — 한번 배워보는 거예요"

**📖 기준 참고사항**
- 건강한 20대 기준 → 나이 들면 가동범위 줄어드는 것 정상
- **좌우 비교가 핵심** — 절대적 기준보다 좌우 차이 위주로

## SECTION 5 | 가쪽굽힘 검사 (Lateral Flexion Test)

**💡 핵심 한 줄**
뒤에서만 검사 — 45도 + 목과 머리가 함께 움직이는지 확인.

**📖 검사 기준**

| 항목 | 기준 |
|---|---|
| 자세 | 뒤에서만 검사 |
| 양적 기준 | 45도 기울어짐 (귀·귀 이은 선 45도) |
| 질적 기준 | 목과 머리가 **함께** 움직여야 함 |
| 양성 | 머리만 따로 움직임 (목은 안 움직이고) |
| 양성 | 45도 미만 or 통증 |

> "머리만 이렇게 돌아가는 경우가 있어요 — 머리와 목이 함께 움직여야 된다는 거 기억해 주시고"

## SECTION 6 | 사각근 ART·MET (Scalenes ART·MET)

**💡 핵심 한 줄**
가쪽굽힘 + 신전 제한, TOS 손저림 — 사각근을 풀어야 목 옆이 열린다.

**📌 언제 해?**

| 적용 상황 | 이유 |
|---|---|
| 가쪽굽힘 제한 | 사각근 타이트 → 가쪽굽힘 방해 |
| 신전 제한 | 사각근 → 신전 방해 |
| TOS (흉곽출구증후군) 손저림 | 사각근 긴장 → 신경 압박 |
| 흉식 호흡 과다 | 사각근 = 2차 호흡근 |
| 목 통증 | 사각근 = C2~C7 횡돌기 부착 |

> 💡 **"목이 아픈데 손이 저린데 보니까 흉식호흡을 과하게 하더라 — 소흉근이랑 사각근 세트로"**

**사각근 특징:**
- C2~C7 횡돌기에 부착 → 1번·2번 갈비뼈로 연결
- 굉장히 얇은 근육
- SCM 뒤쪽에서 앞뒤로 긁으면 다닥다닥 걸리는 느낌 = 사각근

**촉진 방법:**
- SCM 잡고 엄지를 목 방향으로 푹 찔러 넣기
- 앞뒤로 긁으면 밧줄처럼 걸리는 것들 = 사각근

---

### ✂️ 짧아지는 자세 vs 늘어나는 자세

```
짧아진 자세 (시작):
가까워지는 가쪽굽힘 + 약간 굴곡

가장 늘어나는 자세 (끝):
멀어지는 가쪽굽힘 + 머리 살짝 떨구기
```

### 🙌 ART 1 — T-P (횡돌기) 적용
- **책:** 45페이지
- **접촉:** SCM 뒤에서 수직으로 내려가기
  - 유양돌기 약간 뒤쪽 아래쪽에서 시작
  - SCM을 따라 내려가는 게 아니라 **수직으로** 내려가기
  - 엄지 면 전체를 살려서 (포인트로 누르면 너무 아프고 다칠 수 있음)
- **시작 자세:** 가까워지는 가쪽굽힘 + 약간 굴곡
- **동작:**
  - ① 짧아진 자세 (가쪽굽힘 가까워지게)
  - ② 멀어지는 가쪽굽힘 + 머리 살짝 아래로 떨구기
  - ③ 이때 T-P 쪽으로 수직으로 쓸어내리기
  - ④ 다시 짧아진 자세로 → 반복
- **강의 중 표현:** "포인트로 하면 너무 아프고요 다칠 수가 있어서 — 엄지 면을 다 살려서 쭉 길게"

---

### 🙌 ART 2 — 쇄골 안쪽 (갈비뼈 1·2번) 적용
- **접촉:** 쇄골 안쪽 빈 공간
  - 쇄골 안쪽에 손가락 넣어서 면으로 압박
  - 스로틀 당기듯이 엄지를 안쪽으로 파주기
  - 쇄골 라인이 점점 더 선명하게 생김
- **적용 대상:** 한쪽 쇄골 안쪽 공간이 빵빵하게 차있는 분
- **시작 자세:** ART1과 동일
- **동작:**
  - ① 가까워지는 가쪽굽힘에서 시작
  - ② 멀어지는 가쪽굽힘 + 머리 살짝 떨구기
  - ③ 쇄골 안쪽 공간 면으로 압박 유지
  - ④ 다시 돌아오기 → 반복

> "동남아 가서 스쿠터 스로틀 당기듯이 — 손을 엄지를 당겨서 안쪽 쇄골 안쪽 공간을 파주는 거"

### 🔄 MET 1
- **자세:** 누운 자세
- **시작 자세:** ART 끝 자세 (멀어지는 가쪽굽힘 + 머리 떨군 상태)
- **벽:** 목 옆쪽 전체 막기 + 쇄골·갈비뼈 올라오지 못하게 고정
- **큐잉:** "머리 제자리로 돌아오는 힘 주세요"
- **강도:** 숨 참기 + **20~30%**
- **날숨에:** 힘 빼고 → 가볍게 스트레칭

> "가장 멀어진 상태에서 — 제자리로 돌아오는 힘"
