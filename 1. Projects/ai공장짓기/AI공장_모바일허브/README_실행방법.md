# AI 공장 허브 — 실행 방법

방역·꽃집·정부지원사업·RTS 4개 AI 공장의 승인화면 목업을 폰에서 앱처럼
하나로 묶어 여는 허브다. 파일을 복사하지 않고 각 공장 폴더의 원본 파일을
그대로 링크하는 방식이라, 원본을 고치면 허브에도 바로 반영된다.

## 실행 순서

1. **PC에서 `AI공장허브_실행.bat` 더블클릭.** 콘솔 창이 뜨고 로컬 서버가
   시작된다(포트 8899). 자동으로 PC 브라우저에서 허브 화면이 열린다.
2. **콘솔에 뜨는 IP 주소를 확인한다.** 아래 두 줄이 보인다.
   - `PC에서 접속: http://localhost:8899/...`
   - `폰에서 접속(같은 wifi 필요): http://<PC의 IP 주소>:8899/...`
   두 번째 줄의 `<PC의 IP 주소>` 부분을 기억해 둔다(예: `172.20.1.184`).
3. **폰을 PC와 같은 wifi에 연결한다.** (모바일 데이터로는 접속 안 됨 — PC와
   같은 네트워크 안에 있어야 한다.)
4. **폰 브라우저에서 콘솔에 뜬 주소를 그대로 입력해 접속한다.**
   방역/꽃집/정부지원/RTS 4개 카드가 뜨는 홈 화면이 보이면 성공.
5. **"홈 화면에 추가"로 앱 아이콘을 만든다.**
   - iOS(Safari): 하단 공유 버튼 → "홈 화면에 추가"
   - 안드로이드(Chrome): 오른쪽 위 점 3개 메뉴 → "홈 화면에 추가" 또는
     "앱 설치"
6. **이후에는 그 아이콘을 눌러 앱처럼 실행한다.** (단, PC 서버가 켜져
   있어야 열린다 — 아래 한계 참고.)

## 한계

- 이건 진짜 클라우드 배포가 아니다. **PC가 켜져 있고 서버가 실행 중이어야,
  그리고 폰이 PC와 같은 wifi에 있어야만** 동작한다. PC를 끄거나 폰이 다른
  네트워크(예: 카페 wifi, 모바일 데이터)로 옮기면 접속이 안 된다.
- 진짜 "어디서든 접속되는" 앱을 원하면 GitHub Pages 같은 실제 외부
  호스팅이 별도로 필요하다.
