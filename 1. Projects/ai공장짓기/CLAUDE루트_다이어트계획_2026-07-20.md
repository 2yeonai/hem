# 루트 CLAUDE.md 다이어트 계획 (diff 계획만 — 실제 이동 금지) [hyemi, 2026-07-20]

## 제거 금지 예외 (Codex 검수 필수 항목)
- 센티널 3부 규칙 / 잘림 버그 대응(heredoc+wc -c) — 사고 재발 방지 핵심
- 계정 태그 = git config 기준 / 다중계정 동시작업 규칙
- 비개발자 커뮤니케이션 규칙, 명령어 사전 포인터, 상태 정본 2단계 규칙(07-20 신설)
- 꽃집 오사본 RESOLVED는 서사만 이동하되 "꽃집 루트는 진짜 꽃집 스킬" 1줄은 유지
- jbjw-main 중첩 프로젝트 경고 1줄 유지

## 예상 결과
- 18,888자 → 약 8,000~9,000자 (세션당 **약 4~5천 토큰 절감**, 모든 세션에 적용되므로 효과 최대)
- 신설 파일 2개(공장패턴_계약, COMMANDS) + decision-log 이동분 — 삭제는 0건

## 위험
1. 다른 문서가 루트 CLAUDE.md 특정 구역을 참조할 가능성 → 이동 전 `link_scan.py`로 역링크 확인
2. 이동한 규칙을 세션이 못 찾는 문제 → 루트에 링크 1줄씩 반드시 남김
3. 압축 중 의미 변경 → Codex가 이동 전/후 규칙 목록 1:1 대조

## 실행 순서 (승인 후)
① 혜미 승인 → ② link_scan 역링크 확인 → ③ heredoc으로 신설 2파일 작성 → ④ 루트 교체(전문 재작성, wc -c 검증) → ⑤ Codex 규칙 누락 검수 → ⑥ 30일 구조 동결
