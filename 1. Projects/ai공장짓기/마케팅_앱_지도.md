# 가게 홍보·마케팅 앱 지도 (범용 러너 연동 관점, 2026-07-12)

> 지시 4번 기록. 기준: mo,on(가정방문 산후회복) + 향후 꽃집·방역 등 소상공인 공장 공통.
> 각 앱을 "러너에 어떻게 연결하나(연동 방식)"까지 판정해 둠 — 이게 범용러너 문서에 두는 이유.

## 지도 (우선순위순)

| 앱/플랫폼 | 용도 | 연동 방식 | 자동화 판정 |
|---|---|---|---|
| **네이버 스마트플레이스** | 지도 노출·예약·리뷰 관리 — 지역 서비스업 1순위 | 공식 글쓰기 API 없음 → **Chrome 대행** | 리뷰 답글 초안 자동 생성 가능 |
| **인스타그램(+릴스)** | 시각 콘텐츠·지역 고객 | Meta 공식 API 있음(비즈니스 계정) → 장기적으로 API 승격 | 콘텐츠 공장 파생 채널로 추가 가능 |
| **Threads** (@maasittang, @2yeon_sz) | 이미 운영 중 | 공식 API 있음(토큰 60일) — [[1. Projects/클로드 콘텐츠 ai/게시연결_방법조사|기존 조사]] | 예약발행 완전 자동화 후보 1호 |
| **네이버 블로그** (eunoia9496) | 검색 유입·수익화 | API 없음 → Chrome 대행 (확정) | 콘텐츠 공장 이미 연결 설계됨 |
| **카카오톡 채널 + 알림톡** | 고객 소통·예약 알림·재방문 유도 | 공식 비즈니스 API (템플릿 사전 승인제) | 방역·mo,on 방문 안내에 최적 → [[ai공장짓기/카톡_알림봇_방법조사|카톡 조사]] |
| **구글 비즈니스 프로필** | 구글 지도 노출 | 공식 API 있음 | 등록 1회 후 방치 가능 (저비용) |
| **당근비즈니스** | 동네 타깃 광고 | API 없음 → Chrome 대행 | 지역 서비스업(mo,on·방역)에 효과적 |
| **Canva** | 카드뉴스·썸네일 | **Cowork 커넥터 이미 연결됨** | 즉시 사용 가능 — "카드뉴스 만들어줘" |
| **리틀리(link-in-bio)** | 프로필 링크 허브 | Chrome 대행 | 채널 개설 후 1회 세팅 |

- MCP 레지스트리 재확인(2026-07-12): SNS 게시용 커넥터 **여전히 없음** (instagram/naver/kakao 검색 0건) → Chrome 대행이 계속 표준.

## 러너 관점 요약
- 콘텐츠 공장 manifest의 게시 stage는 채널이 늘어도 구조 동일 — channel_config.yaml에 행만 추가하면 됨(인스타·플레이스 리뷰답글 등).
- 마케팅 공장을 따로 지을 필요 없음: **콘텐츠 공장 확장 + 알림톡(카톡 조사 문서)** 두 개로 위 표 전부 커버.
