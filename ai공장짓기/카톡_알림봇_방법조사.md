# 카톡 자동화 + 모바일 알림봇 방법조사 (2026-07-12)

> 지시 9번: "안 된다 하지 말고 방법 강구해서 기록" — 결론: **둘 다 가능. 아래 방법 중 골라서 30분이면 시작.**

## A. 모바일 알림봇 (작업이 멈추지 않게 폰으로 승인 요청 받기)

목표 예시: 폰에 "혜미님 ㅇㅇ 검수 or 동의는 필수입니다. 확인해주세요" 푸시가 옴.

| 방법 | 설치 난이도 | 원리 | 판정 |
|---|---|---|---|
| **① ntfy (추천 1순위)** | ★☆☆ 5분 | 폰에 ntfy 앱 설치 → 나만의 토픽 구독 → Cowork가 인터넷 요청 한 줄로 푸시 발송 | 계정·토큰도 필요 없음. 가장 빠름 |
| ② 텔레그램 봇 | ★★☆ 15분 | 텔레그램에서 BotFather로 봇 생성(혜미 1회) → 토큰으로 Claude가 메시지 발송 | 대화형(답장으로 "승인" 회신)까지 확장 가능 |
| ③ 카카오 "나에게 보내기" API | ★★★ | 카카오 개발자 앱 등록 + 토큰 갱신 관리 | 카톡으로 받고 싶으면 이것. 관리 부담 있음 |

**추천 조합**: 지금은 ①ntfy로 시작(오늘이라도 가능) → 답장 승인까지 원하면 ②텔레그램 승격.
**혜미 몫**: 앱 설치 + 토픽 이름 1개 알려주기. 끝.

## B. 카톡 자동화 (요약 → 생각 → 답장)

| 방법 | 가능 범위 | 리스크/조건 |
|---|---|---|
| **① 카톡 공식 AI 요약** | 긴 대화 요약 (PC톡: 말풍선 우클릭 → "여기서부터 요약") | 즉시 사용 가능, 리스크 0 |
| **② PC톡 내보내기 → Cowork 루프 (추천)** | 대화 내보내기 → Claude가 요약·답장 초안 생성 → 혜미 승인 → PC톡으로 발송. 컴퓨터 켜져 있으면 예약작업으로 반복 가능 (실사례 검증됨: Threads @tofukyung 사례) | 개인 계정 안전. 발송 직전 승인 게이트 유지 |
| ③ 카카오톡 채널 + 알림톡 | **고객용** 공식 자동 발송(예약 안내, 방문 리마인드) — mo,on·방역에 최적 | 비즈니스 채널 개설 + 템플릿 사전 승인 필요 |
| ④ 안드로이드 알림 리스너 봇 (kakao-talk-auto-bot 등 오픈소스) | 지정한 방만 AI 자동 답장 | 폰 상시 구동. **개인 카톡 자동발송은 계정 제재 위험 → 완전 무인 금지, 게이트 유지** |
| ⑤ MAKE 등 노코드 | 대화 정리 → 뉴스레터/타채널 발송 | 구독료. 우리 러너로 대체 가능 |

**설계 결론 (company_charter 정합)**: ②+③ 조합이 정답.
- 개인 카톡 = ② (요약·초안 자동, **발송만 승인 후**)
- 고객 카톡 = ③ 알림톡 (승인된 템플릿은 완전 자동 발송 가능 — 법적으로도 안전한 유일한 완전자동 경로)
- 승인 요청 알림은 A의 ntfy/텔레그램으로 폰에 도착 → 모바일에서 작업이 멈추지 않음.

## 다음 액션 (혜미가 "카톡봇 시작해줘"라고 하면)
1. ntfy 앱 설치 안내 → 토픽 연결 테스트 발송 (Haiku급, 10분)
2. PC톡 내보내기 폴더 지정 → 요약·답장초안 예약작업 등록 (Sonnet, 30분)
3. (사업용) 카카오톡 채널 개설은 혜미 본인 인증 필요 — 개설되면 알림톡 템플릿 초안은 Claude가 작성
