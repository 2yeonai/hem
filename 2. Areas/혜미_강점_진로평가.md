# 혜미 강점 분석 + 진로 평가 (Fable 5, 2026-07-12) #진로

> 질문: "출장 산후마사지(손발) vs 바이브코딩(컴퓨터) vs 정부지원 컨설턴트 — 뭐가 가장 큰 이득?"
> 판정 기준 3개: 확실성(성공 확률) × 자산 정합(이미 가진 것과 맞나) × 확장성(시간 안 늘려도 커지나)

## 1. 관찰된 강점 (지난 5일 실작업 근거)
1. **시스템 사고** — 비개발자인데 계약·승인게이트·파이프라인 개념을 체득 (증거: 이 볼트 구조 자체. 이건 배워서 되는 게 아니라 드묾)
2. **실행 지구력 + 기록 문화** — 3일 만에 러너·콘텐츠 공장·자동기록까지 구축, 끊겨도 HANDOFF로 복원
3. **도메인 자산** — 미용(피부) 국가자격 + 산후 회복 **당사자 경험** + 진주 지역 기반 (AI가 절대 대체 못 하는 3종)
4. **콘텐츠 감각** — @2yeon_sz 바이럴 실적 (검증된 수요 발굴 능력)
5. 보완점: 판단 과부하(수집≫처리 — 이미 파일 위임으로 해결 중), 코드 품질 자가검증 불가(검증자 세션으로 보완), 트렌드에 세팅 흔들림(그만할 것 1번)

## 2. 세 갈래 평가
| | A. 출장 산후회복 (mo,on, 손발) | B. 바이브코딩 (주업) | C. 정부지원 컨설턴트 |
|---|---|---|---|
| 확실성 | ★★★ 자격+경험+수요 검증됨 | ★ 시장 포화, 품질 자가검증 불가 | ★★ 본인 선정 실적 이후엔 ★★★ |
| 자산 정합 | ★★★ 3종 자산 전부 사용 | ★★ 시스템 사고만 사용 | ★★★ 이미 도구(8인심사위원) 보유 |

## 3. 판정 — 택일이 아니라 순서다
**"AI로 무장한 현장 사업자"가 혜미의 유일무이한 포지션.** 손발(A)과 컴퓨터(B·C)는 경쟁 관계가 아니라, 컴퓨터가 손발 사업의 마진과 시간을 만들어주는 구조. 셋 다 하는 사람이 시장에 거의 없음 — 그게 해자.

1. **지금~8월: A가 앞** — mo,on 신청서(실행계획 3단계)가 최고 기대수익(지원금 수천만 + 본업 개시). A의 약점(시간=매출)은 AI공장이 예약·기록·콘텐츠를 대신해 완화.
2. **선정 후: C 개시** — "내가 내 신청서를 AI로 써서 선정됐다"가 최강 영업 자료. 무료 1~2건(지인 소상공인) → 유료. 컨설팅은 A의 현장 신뢰 위에서만 팔림.
3. **B는 주업 금지, 레버리지로** — 바이브코딩은 A·C를 싸게 만드는 내부 무기 + 검증된 것만 소액 판매(수익화 M4). 주업으로 삼으면 강점(도메인)을 버리고 약점(코드 검증)으로 싸우는 것.

## 4. 이 판정이 틀렸다는 신호 (이게 보이면 재평가 요청)
- mo,on 첫 3개월 예약이 주 2건 미만 → A 축소, C·콘텐츠 비중 상향
- 컨설팅 무료 건이 선정 실패 → C 보류, 도구 개선 먼저
- @2yeon_sz 수익화가 월 50만+ 안정 → 콘텐츠 비중 상향 검토
