# usage.md — 사용량 신호등

> ⚠️ 재구성 안내: 2026-07-15 세션 원본 유실로 인한 v1 재구성본. "신호 격상 규칙"의 정확한 임계값은 세션로그에 남지 않아 임의 기본값을 채움 — 실제 사용하면서 혜미 판단으로 조정할 것.

## 신호등 상태

| 신호 | 뜻 | 지금 해야 할 일 |
|---|---|---|
| 🟢 초록 | 여유 있음 | 평소대로 진행 |
| 🟡 노랑 | 한도 근접 | 지금 하던 작업 하나만 마무리하고 새 작업 시작 자제 |
| 🔴 빨강 | 한도 도달/근접 초과 | 즉시 "어디까지 했는지" 세션로그에 기록하고 중단 — [[ai공장짓기/CLAUDE.md|모델 사용 기준]] 4번 규칙 |

## 신호 격상 규칙 (임의 기본값 — 확인 필요)
- 🟢→🟡: 세션 한도의 약 70% 지점(체감 기준, 정확한 %를 시스템이 알려주지 않는 도구가 많아 "체감"으로 판단)
- 🟡→🔴: 한도 도달 경고가 뜨는 시점, 또는 응답이 눈에 띄게 느려지거나 끊기는 시점
- 🔴 상태에서 새 작업을 시작하지 않는다 — 반드시 기록 후 중단

## 실물 기록 예시
```
[2026-07-15 14:20] 🟡 진입 — AI운영체제 설계서 2단계 정합 작업 중, 8개 파일 중 5개 완료
[2026-07-15 14:35] 🔴 진입 — 즉시 세션로그에 진행상황 기록 후 중단
```
