1. 개발 목표
&#8728; 디자인적 차별화와 실용성을 가지는 학교 생활 및 교육 플랫폼 UX 개발
- 직관적인 사용방법 및 정보확인이 가능한 화면 UX 개발
- 타 학교 생활 앱과 차별성을 가지는 요소 적용
- 각 분류별(교사, 학생, 학부모(일반)) 적절한 UX 요소 적용

&#8728; 하이브리드 앱 기획 및 개발
- Android, IOS, WEB에서 구동이 가능한 크로스플랫폼 앱 설계 및 개발
* 최적화
교사 &#8211; WEB
학생 및 학부모(일반) &#8211; Android, IOS

&#8728; 통합 시스템에 적합한 DB 설계
- 세 분류(교사, 학생, 학부모(일반))을 고려한 설계 진행
- 교육, 학교 생활 등 관련 콘텐츠 내용 설계 진행
- 유지·보수가 용이하고 생산이 효율적인 구성
- 높은 품질 반영 및 A/S가 용이한 구조

&#8728; Product Identity 구축
- 유행을 타지 않는 ‘Timeless design’ 컨셉의 Product Identity 구축
- 사용자 친화적이며 독창적인 개발을 통한 제품경쟁력 향상
- 제품의 목표시장과 고객층에 따른 맞춤형 개발 진행(웹, 앱, 등)

4. Spec.
1) 개발 언어 및 버전
- Java 11

2) 데이터베이스
- 종류: PostgreSQL
- 버전: 42.3.1

3) 프레임워크 및 라이브러리
- Spring Boot: 2.7.13
- Lombok, Spring Data JPA, Spring Cloud AWS, Springfox 3.0.0

4) 개발 도구
- 빌드 도구: Gradle
- 코드 버전 관리 : Git

5) 배포 환경
- 서버: AWS EC2
- OS 버전 : Amazon Linux
- CI/CI 도구 : GitHub Actions

6) 기타 사용한 서비스 또는 기술
- AWS 관련 서비스 (RDS, S3, CloudWatch, CodeDeploy, Budget)
