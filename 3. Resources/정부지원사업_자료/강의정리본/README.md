## 초창패 전자책 (원본 강의 교재 — 미변환 상태)
"초기창업패키지" 강의 원문 교재. `s1`~`s9` 챕터 + `0`/`00`(서문 추정) + `s0~s9.pdf`(합본) + `초창패 전자책.hwp`(전체본), 전부 hwp/pdf 원본만 있고 **다른 자료들과 달리 `_원문/` md 변환이 안 되어있음**(이 폴더 상위 `정부지원사업_자료/_원문/`에는 이 시리즈가 없음).

위 1~4강 정리본이 이 전자책 내용을 재구성한 강의 요약일 가능성이 높음(같은 초기창업패키지 강의 계열) — 다만 챕터(s1~s9)와 정리본(1~4강)의 정확한 대응관계는 확인 안 됨. 필요하면 s1~s9 원문도 `_원문` 변환 진행 가능.
