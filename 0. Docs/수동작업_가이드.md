# 수동작업 가이드 — 혜미가 직접 눌러야 하는 것 전부 (아주 쉽게) #지침

> 여기 있는 것만 사람 손이 필요하고, 나머지는 전부 클로드가 함. 각 항목 5분 이내.

## 1. 파이썬 설치 (러너 웹앱 쓰려면 1회 필요)
파이썬 = 러너 앱을 움직이는 엔진. 무료·안전(공식 사이트).
1. 인터넷 주소창에 **python.org** 입력 → 노란색 **Download Python** 버튼 클릭
2. 받은 파일 더블클릭 → 설치 첫 화면 **맨 아래 "Add python.exe to PATH" 네모칸에 꼭 체크** ← 이거 하나가 제일 중요
3. **Install Now** 클릭 → 끝나면 Close
4. 확인: 러너앱_실행.bat(아래 2번)이 열리면 성공

## 2. 러너 웹앱 켜기 (공장을 버튼으로 돌리는 화면)
1. 바탕화면 → **hem 폴더** 열기 → **러너앱_실행.bat** 더블클릭
2. 검은 창이 뜨고 브라우저가 자동으로 열림 (안 열리면 주소창에 `localhost:8765`)
3. 공장 목록에서 **[검증]**(설계도 검사) 또는 **[모의실행]**(끝까지 돌려보기) 클릭 → 결과가 아래 그대로 표시됨
4. 끄기: 검은 창 닫기

## 3. 3D 그래프 보기 (볼트를 우주처럼 날아다니기)
1. hem 폴더 → **도구** 폴더 → **옵시디언_3D그래프.html** 더블클릭 (인터넷 연결 필요)
2. 마우스 드래그=회전, 휠=확대, W/S=전진/후진, 점 클릭=노트 미리보기
3. 노트가 늘어난 뒤 갱신하려면: 코워크에 "3D 그래프 다시 만들어줘" 한 줄

## 4. 커넥터 연결 (전부 클릭+로그인만)
- **Figma**: 클로드 설정 → Connectors → Browse → "Figma" 검색 → Connect → 로그인. (용도: 디자인 시안을 클로드가 읽고 카드뉴스·앱 화면 규격 추출)
- **Higgsfield**(AI 사진·영상): 설정 → Connectors → **커스텀 커넥터 추가** → 주소칸에 `https://mcp.higgsfield.ai/mcp` 붙여넣기 → 추가 → 로그인 (Higgsfield 구독 필요)

## 5. 이전 채팅 백업 (계정 기록 가져오기)
claude.ai → 설정 → Privacy → **Export data** → 메일로 온 파일을 hem 폴더의 `_inbox`에 넣기 → 끝 (밤에 자동 분류됨)

## 6. 예약 작업 확인 (자동기록이 도는지)
클로드 앱 왼쪽 사이드바 → **Scheduled** → daily-session-log, weekly-synthesis 보이면 정상. 각각 **Run now** 한 번씩 눌러두면 이후 권한 질문 없이 자동으로 돎.

## 7. 뭔가 이상할 때 만능 문장
```
[뭐가 이상한지 한 줄]. 고치지 말고 먼저 원인만 진단해서 증거와 함께 보여줘.
```

## 8. 카카오톡 연결 — "나에게 카톡 보내기" (1회 설정)
카카오 공식 클로드 커넥터는 아직 없음(2026-07-12 레지스트리 확인). 카카오가 운영하는 **PlayMCP**를 커스텀으로 연결:
1. 클로드 설정 → Connectors → **커스텀 커넥터 추가** → 브라우저에서 **playmcp.kakao.com** 접속해 안내되는 연결 주소를 복사해 붙여넣기 → 카카오 로그인
2. 연결되면 아무 세션에서 테스트: `"나에게 카톡 보내줘: 연결 테스트 성공!"` → 내 카톡 나와의채팅방에 도착하면 성공
3. ⚠ 한계: 본인 인증 기반이라 예약 자동발송은 불안정할 수 있음(베타) — 중요 알림용으론 쓰되 계약·결제 관련 금지
