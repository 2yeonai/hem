# 클로드 코드 데스크탑 사용 지침

> 이 문서는 "클로드 코드 데스크탑 앱을 어떻게 켜고 쓰는가"만 다룹니다. 오늘 뭘 할지 순서는 [[0. Docs/혜미_클로드_사용설명서|혜미_클로드_사용설명서]]를 보세요.

## 1. 클로드 코드 데스크탑이 뭔가요

**클로드 코드**는 채팅창처럼 대화하지만, 내 컴퓨터(이 hem 폴더) 안의 파일을 직접 읽고 고치고 저장할 수 있는 클로드입니다. 인터넷 너머 어딘가가 아니라 **내 PC 안에서 직접 돌아갑니다.**

**Cowork(웹 버전)와 다른 점**: Cowork은 클로드 회사의 다른 컴퓨터(샌드박스)에서 돌아가서, 큰 폴더 작업이 느려지거나 깃허브 저장(푸시)이 막히는 문제가 있었습니다(2026-07-17 실제로 발생). 클로드 코드는 내 PC에서 바로 돌아가므로 이 문제가 없고, git 저장(커밋·푸시)도 바로 됩니다.

## 2. 작업 시작하는 법

1. 클로드 코드 앱을 엽니다.
2. 화면 상단이나 제목표시줄에 폴더 경로가 **`C:\Users\82106\Desktop\hem`**로 되어 있는지 확인합니다. 다른 경로면 그 창에서 폴더 열기로 hem 폴더를 선택하세요.
3. 하던 작업을 이어가려면 그냥 **"작업이어서해줘"** 한 마디만 치면 됩니다 — 클로드가 알아서 최근 세션로그를 읽고 이어갑니다.
4. 새로운 일을 시키고 싶으면 그냥 하고 싶은 말을 그대로 치면 됩니다(예: "방역 골든셋 검증해줘").

## 3. 모델 고르는 법

- 입력창에 **`/model`**을 치면 고를 수 있는 목록이 뜹니다.
- 평소엔 **Sonnet**(기본값, 그대로 두면 됨)으로 충분합니다.
- **Fable**은 값이 비싼 모델이라, 설계를 새로 정하거나 뭔가 서로 부딪힐 때만 씁니다. 클로드가 필요하다고 판단하면 먼저 "Fable 쓸까요?"라고 물어보게 되어 있으니, 혜미가 미리 고를 필요는 없습니다.

## 4. 권한 팝업이 뜨면

클로드가 파일을 고치거나 컴퓨터에서 명령을 실행하기 전에 "이거 해도 돼?"라고 묻는 창입니다.

- 파일 읽기·쓰기, git 저장(커밋) 같은 건 보통 **허용**해도 됩니다.
- **절대 시키면 안 되는 것** (클로드가 요청해도 거절): 비밀번호·카드번호 입력, 돈 보내기·송금, 계정 새로 만들기 — 이런 건 항상 혜미가 직접 해야 합니다.

## 5. 토큰이 떨어지면 (계정 전환)

"토큰"은 클로드를 쓸 수 있는 사용량입니다. 다 쓰면 다른 계정으로 바꿔야 합니다.

1. **"기록 남겨줘, 다음 계정으로 넘어갈게"**라고 말합니다 → 클로드가 세션로그에 어디까지 했는지 기록하고 저장까지 합니다.
2. 앱에서 지금 계정으로 로그아웃합니다.
3. 다른 계정으로 로그인합니다.
4. 새 세션에서 **"작업이어서해줘"**라고 치면, 방금 남긴 기록을 읽고 이어갑니다.

## 6. 저장(커밋)은 이제 자동 검사됨

2026-07-18부터, 저장(커밋)할 때 규칙을 어기면 **자동으로 저장이 막히도록** 설정했습니다(마치 규칙에 안 맞는 서류는 도장이 안 찍히게 만든 것과 같습니다). 클로드가 알아서 처리하므로 혜미가 따로 신경 쓸 건 없습니다. 화면에 **"저장이 막혔다"**는 메시지가 보이면, 클로드에게 그대로 알려주면 클로드가 원인을 고치고 다시 저장합니다.

## 7. PowerShell vs cmd 구분법

클로드가 "터미널에서 실행하세요"라고 하면 아무 검은 창이나 열면 안 되고, 어떤 걸 열지 구분해야 합니다.

- **PowerShell** (이 볼트 작업은 거의 다 이걸 씁니다): 시작 버튼 → "PowerShell" 입력 → 파란/검은 아이콘의 창이 뜨면 그것.
- **cmd** ("명령 프롬프트"): 시작 버튼 → "cmd" 입력 → 검은 창. 클로드가 명시적으로 "cmd 쓰세요"라고 할 때만 이걸 엽니다.
- 헷갈리면 클로드에게 **"PowerShell이야 cmd야?"**라고 되물으면 바로 알려줍니다.

## 8. 문제가 생기면

- 에러 메시지든 이상한 화면이든, 그냥 **화면을 그대로 복사해서** 클로드에게 붙여넣으면 됩니다.
- 무슨 뜻인지 모르겠으면 **"이게 무슨 뜻이야?"**라고 물으면 쉬운 말로 풀어서 설명해 줍니다.
