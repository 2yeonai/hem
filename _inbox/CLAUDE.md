# _inbox — 자동 반입용 폴더

## 목적
claude.ai 웹 채팅에서 다운받은 파일(md/zip)을 여기 넣으면, 매일 밤 예약 작업이 자동으로 올바른 폴더에 분류한다. 혜미가 할 일: 다운로드 시 저장 위치를 이 폴더로 지정 (드래그 한 번).

## 자동 분류 규칙 (예약 작업이 따를 것)
1. zip이면 압축 해제 후 md만 처리, 원본 zip은 _inbox/처리완료/로 이동.
2. 내용 주제를 보고 이동: AI공장 설계→ai공장짓기/, 공장별 자료→해당 공장 폴더, 참고자료→3. Resources/, 지식노트→5. Zettelkasten/00. Inbox/.
3. 같은 주제 노트가 이미 있으면 새 파일 만들지 말고 기존 노트에 내용 추가 (볼트 규칙).
4. 삭제 절대 금지 — 이동만. 처리 결과는 그날 세션로그 노트에 한 줄씩 기록.
5. 판단이 안 서는 파일은 건드리지 않고 세션로그에 "분류 보류"로 보고.
