## 이번 달 사용 가능 AI

/ai에서 체크한 AI만 라우팅된다. 실패하면 다음 체크된 AI로 폴백, 전부 실패·미체크면 로컬 결과로 진행. 1개만 체크하면 그 AI로 고급 작업 전부 수행. **결제 기능 없음** — 비용은 각 AI 계정에서 결제된다.

## API 키 보안

- 입력 즉시 사용 가능, 저장은 선택
- 평문 저장 금지 — **사용자 비밀번호 기반 암호화**(PBKDF2 20만회 + 무결성 MAC)로 `00_local/ai_keys.enc`에 저장 (git 제외)
- 다음 실행 시 비밀번호 입력으로 복호화. **비밀번호 분실 시**: /ai → "저장 키 삭제" 후 재입력
- 키는 화면·로그·Export 어디에도 노출되지 않는다 (마스킹)
