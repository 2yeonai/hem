# replacement_words.md — 대체 표현 사전

원칙: 자신감과 방어 가능성의 균형. 성장성은 남기고 단정은 없앤다.

| 위험 표현 | 대체 표현 |
|---|---|
| 완벽히 해결 | ~의 위험을 줄임 / 오류를 완화함 |
| 100% 가능 | 가능성을 높임 / 수행기간 내 가능한 범위에서 실행 |
| 무조건 가능 | 단계적으로 검증하며 진행 |
| 자동으로 모두 처리 | 자동 생성 후 검수하여 처리 / 검수 후 진행 |
| 검증 완료 | 시범운영을 통해 검증 예정 / 초기 검증 결과 확보 |
| 확정 (제휴·판매) | 협의 중 / 구매의향서 확보 후 추진 / 증빙 확보 후 추진 |
| 독점 | 초기 진입 우위 확보 |
| 최고 수준 | 목표 품질 기준 설정 후 관리 |
| 세계 최초 | 기존 방식과의 차별점: [구체 항목] |
| 실패하지 않음 | 위험 발생 시 보완계획 보유 |
| 효과 보장 | 초기 반응 확보 후 단계적 확대 |
| 대규모 확장 | 단계적 확대 / 현재 단계에서는 [범위]까지 실행 |
| AI가 판단 | AI가 초안을 생성하고 신청자가 검수·확정 |
| 자동 발송 | 발송 목록 자동 생성 후 확인하여 발송 |
| 무인 운영 | 운영 시간을 단축하는 반자동 운영 |

## 안전 문장 패턴

- "현재 단계에서는 [범위]까지 실행하고, 결과를 바탕으로 다음 단계에서 보완하겠습니다."
- "수행기간 내에는 [산출물]을 완성하고, [확장]은 후속 계획으로 추진합니다."
- "확보된 증빙은 [A]이며, [B]는 수행기간 중 확보 예정입니다."
