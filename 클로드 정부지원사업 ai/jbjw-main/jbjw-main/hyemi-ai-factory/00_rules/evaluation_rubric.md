# evaluation_rubric.md — 공장 내부 채점 기준

주의: 실제 채점은 항상 **현재 공고의 심사표가 우선**한다. 이 문서는 심사표가 없거나 내부 검수할 때 쓰는 기본 채점틀이다.

## 아이디어 후보 평가 (5점 척도)

| 평가축 | 5점 | 3점 | 1점 |
|---|---|---|---|
| 공고 적합성 | 공고 목적과 직접 일치 | 부분 일치 | 목적과 어긋남 |
| 배점 적합성 | 배점 높은 항목에서 강함 | 보통 | 배점 낮은 항목만 강함 |
| 문제 명확성 | 시간·비용·오류로 수치화 가능 | 서술은 되나 수치 없음 | 감성 서사뿐 |
| 고객 명확성 | 첫 고객·수혜자 특정 가능 | 범위만 존재 | 불명확 |
| 증빙 가능성 | 확보 증빙 다수 | 일부 확보·일부 예정 | 증빙 없음 |
| 실현 가능성 | 수행기간 내 직접 실행 가능 | 일부 외부 의존 | 기간 내 불가능 |
| 차별성 | 기존 대안 대비 구체적 차이 | 부분적 차이 | 동일 대안 존재 |
| 예산 타당성 | 항목-산출물-성과 연결 | 일부 연결 | 단순 구매·외주로 보임 |
| 발표 방어 | WHY 3종 답변 가능 | 일부 답변 가능 | 방어 불가 |
| 리스크 | 결격·감점 요소 없음 | 관리 가능한 리스크 | 탈락급 리스크 |

점수 기준: 5 강함 / 4 비교적 강함 / 3 보통 / 2 약함 / 1 위험하거나 부적합

## 내부 검수 채점 (100점, 심사표 없을 때)

| 항목 | 배점 |
|---|---|
| 문제 인식·필요성 | 20 |
| 해결방안·차별성 | 20 |
| 실행계획·실현 가능성 | 25 |
| 예산 적정성·산출물 연결 | 15 |
| 신청자 역량·증빙 | 20 |

## 합격 기준

- 내부 검수 80점 미만이면 제출 금지, revision_order 생성
- 탈락급 리스크(자격 미달, 지원제외, 허위 기재 위험) 1건이라도 있으면 점수 무관 STOP
