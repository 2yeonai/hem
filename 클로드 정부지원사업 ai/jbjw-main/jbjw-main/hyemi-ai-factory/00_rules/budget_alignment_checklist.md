# budget_alignment_checklist.md — 예산 정합성 체크리스트

원칙: 예산은 지출 목록이 아니다. **예산 항목 → 사용 목적 → 산출물 → 성과목표 → 평가항목 연결**이 한 줄로 보여야 한다.

## 항목별 점검 (예산 항목마다 반복)

- [ ] 공고상 허용되는 항목인가 (공고 예산 기준 대조)
- [ ] 이 예산이 왜 필요한지 한 문장으로 답할 수 있는가
- [ ] 어떤 활동·산출물과 연결되는가 ("이 돈을 쓰면 뭐가 남는가")
- [ ] 어떤 성과목표와 연결되는가
- [ ] 금액 산정 근거가 있는가 (견적서, 단가×수량)
- [ ] 수행기간 내 집행 가능한가

## 위험 패턴 점검

- [ ] 단순 구매성으로 보이지 않는가 (장비·물품 비중 과다)
- [ ] 외주 의존으로 보이지 않는가 (외주개발비가 총액의 절반 이상이면 경고)
- [ ] 구독료·범용 소프트웨어 구매로 보이지 않는가
- [ ] 자부담 조건 충족하는가
- [ ] 가족·특수관계자 거래가 없는가
- [ ] 지원제외 비용(공고 명시)이 없는가

## 예산표 필수 형식

| 예산 항목 | 사용 목적 | 산출물 | 산정 근거 | 연결 평가항목 | 기대성과 |
|---|---|---|---|---|---|

## 예산 방어 답변 구조 (발표용)

예산 항목 → 사용 목적 → 산출물 → 성과목표 → 평가항목과의 연결

## 감액 대비

- [ ] "예산을 줄이면 무엇을 조정할 것인가"에 대한 답변이 준비돼 있는가 (우선순위 낮은 항목 지정)
